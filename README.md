The Grouse Gym Management System 🏋️‍♂️

A comprehensive Full-Stack SaaS application designed to streamline gym operations, member tracking, and payment management. Built with a focus on performance and a modern "Glassmorphism" UI.

🚀 Live Demo
Frontend: https://the-grouse-gym.vercel.app/

🛠 Tech Stack

Frontend: - React.js (Vite)
- Redux Toolkit (State Management)
- Tailwind CSS (Styling with Glassmorphism)
- Axios (API Calls)

Backend:
- Node.js & Express
- TypeScript
- PostgreSQL (Database)
- JWT (Authentication)

✨ Features

- Dashboard: Real-time stats on active members and monthly revenue.
- Member Management: Full CRUD functionality for gym members.
- Membership Plans: Flexible plan creation and assignment.
- Payment Tracking: Monitor payment history and pending dues.
- Secure Auth: Role-based access control using JWT.

💻 Getting Started

Prerequisites
- Node.js (v18+)
- PostgreSQL installed and running locally.
